# 🌍 Travel Planner - Planejador de Viagens Inteligente

Um planejador de viagens moderno e elegante com suporte a múltiplos idiomas, galeria de fotos, cápsulas do tempo e muito mais.

## ✨ Principais Funcionalidades

### 📋 Gerenciamento de Viagens
- **Criar e organizar viagens** com datas, locais e imagens de capa personalizadas
- **Timeline de eventos** com detalhes completos (voos, hotéis, atividades, restaurantes)
- **Countdown em tempo real** para próximas viagens
- **Status automático**: Próximas, Em andamento, Concluídas

### 💾 Memórias
- **3 tipos de memórias**: Notas, Fotos e Dicas
- **Busca avançada** em todas as memórias
- **Filtros inteligentes** por tipo (notas, fotos, dicas, cápsulas)
- **Upload de fotos** com legendas

### 🕰️ Cápsulas do Tempo
- Transforme memórias em **cápsulas do tempo**
- Configure uma data de abertura futura
- Conteúdo bloqueado até a data escolhida
- Badge visual indicando status (aberta/bloqueada)

### 📸 Galeria de Fotos
- **Masonry Grid** responsivo para exibição de fotos
- **Lightbox** com navegação por teclado (← →)
- Visualização ampliada com detalhes
- Contador de fotos

### 🌐 Internacionalização (i18n)
- **3 idiomas suportados**:
  - 🇧🇷 Português (Brasil)
  - 🇺🇸 English (Estados Unidos)
  - 🇪🇸 Español (España)
- Tradução completa de toda a interface
- Formatação de datas localizada

### 🎨 Personalização
- **4 temas de cores**:
  - Teal (Padrão)
  - Roxo
  - Azul
  - Rosa
- **Onboarding inicial** para configurar preferências
- **Configurações** acessíveis a qualquer momento

### 🔗 Compartilhamento
- **Links públicos** para viagens
- **Export .ics** para adicionar ao calendário
- Página de preview com visual cinematográfico
- Um clique para copiar link

### 🌙 Design
- **Tema dark elegante** com gradientes
- **Covers circadianos** que mudam com o horário
- **Microinterações suaves** com Motion/Framer Motion
- **Totalmente acessível** (WCAG AA)
- **Responsivo** para todos os dispositivos

### 💪 Funcionalidades Técnicas
- **PWA-ready** (pode ser instalado no celular)
- **Persistência local** com localStorage
- **Modo Demo** com dados seed bonitos
- **Performance otimizada** com lazy loading

## 🚀 Como Usar

1. **Primeiro acesso**: Complete o onboarding escolhendo seu nome, idioma e tema
2. **Criar viagem**: Clique em "Nova Viagem" e preencha os detalhes
3. **Adicionar eventos**: Na página da viagem, adicione eventos à timeline
4. **Registrar memórias**: Crie notas, dicas ou faça upload de fotos
5. **Cápsulas do tempo**: Ative a opção ao criar uma memória
6. **Galeria**: Acesse todas as suas fotos em um grid visual
7. **Buscar**: Use a barra de busca para encontrar memórias específicas
8. **Filtrar**: Filtre por tipo de memória com um clique
9. **Compartilhar**: Gere um link público da sua viagem
10. **Personalizar**: Acesse Configurações para mudar idioma ou tema
